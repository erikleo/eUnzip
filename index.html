eUnzip

The fastest and easiest way to unzip downloads

  • Download ZIP

  • View On GitHub

This project is maintained by erikleo

Purpose

eUnzip removes the frustration and wasted time when unzipping files, by doing the things automatically that should have been done in the first place

Features

By simply right clicking on a zip file and using “send to” -> eUnzip.exe the program will do the following:

  1. Unzip the file
    • If the zip is a bunch of files it will create a folder and put the files in it (removing the possibility of accidental filling your desktop with files all over the place)
    • If the zip is a single folder or file it will just unzip it and not create an extra folder, you don’t need
  2. Delete the original zip file. (which you don’t need anymore)

Directions

  1. Download eUnzip.exe and save it to disk
  2. Copy eUnzip to the windows "Send To" folder
    • To find this folder try the shortcut "Windows Key-> R" then type the following in the run box: "%APPDATA%\Microsoft\Windows\SendTo\"
  3. Once the file is copied, all you need to do is right click on a zip file then choose "Send To", then "eUzip.exe" and the program will unzip the file for you without creating any clutter.

Author Note

I find people almost always want the unzipping process to work this way but every unzip program I found does not do it, so I made one that does. Enjoy!
